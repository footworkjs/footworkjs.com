Roadmap

Where is footwork going? This is the current (incomplete, non-prioritized) list of wants:

  • Custom binding for viewModels.
  • Custom binding for routers.
  • Routerless outlets.
  • <component /> runtime composition, see this thread on github.
  • Named routes with defined/named (object based) parameter passing.
  • Modularized features to allow for custom builds
  • Ready-to-go environments (amd, globals, commonjs)
  • Some method of overall app initialization detection (ie: events triggered depending on what components have loaded).
  • Create sub-router example/tutorial
  • Please feel free to contribute by making a suggestion at github issues!
