Install via bower.json:

If you are totally new to bower, it is recommended that you head over to the bower website to get info on what it is and how to use it.

Generally you will be installing footwork via bower in the context of a larger application and use the bower.json file:

// add footwork under "dependencies"
{
  "dependencies": {
    "footwork": "*"
  }
}
# Then in the same directory as your bower.json file
bower install

You can also optionally install just footwork with the following command-line statement:

bower install footwork
