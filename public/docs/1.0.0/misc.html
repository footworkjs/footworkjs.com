Utility

Utility functions / methods.

  • fw.start() - Start footwork / initialize application
  • fw.utils.guid() - Generate a random GUID.

fw.start()

fw.start( targetElement )

Initialize / start footwork (optionally on the specified node/element):

// start application (defaults to body element)
fw.start();

// start application on #appContainer element
fw.start(document.getElementById('appContainer'));

NOTE: This is very similar to calling fw.applyBindings(). In fact they are identical except fw.start() is a simplified version which does not accept a viewModel to bind, it simply causes footwork to begin processing the DOM (and thus it will initialize any viewModel/router/component/etc definitions it finds in the markup).

fw.applyBindings() will also cause footwork to begin processing the DOM, but it also requires that you supply a viewModel to bind with the root node.
