Utility

Utility functions / methods.

fw.start()
fw.start( targetElement )
Initialize binding / start FootworkJS (optionally on the specified node/element).
// start application (defaults to body element)
fw.start(); // document.body is the default element

NOTE: This is very similar to calling fw.applyBindings(). In fact they are identical except fw.start() is a simplified version which does not accept a viewModel / dataModel / router to bind, it simply causes FootworkJS to begin processing the DOM (and thus it will initialize and compose any <viewModel> / <dataModel> / <router> / <component> declarations it finds in the markup).

fw.applyBindings() will also cause Footwork to begin processing the DOM, but it also requires that you supply a viewModel / dataModel / router to bind with the root node.

fw.utils.guid()
Generate a random GUID (format: ########-####-####-####-############).
Returns: 'string' The name of the namespace channel 
// generate a random GUID
var guid = fw.utils.guid();
// guid === "629e4dfa-fd39-d756-b830-563af555a03f"
