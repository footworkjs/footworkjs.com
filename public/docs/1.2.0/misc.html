Utility

Utility functions / methods.

fw.start()
fw.start( targetElement )
Initialize binding / start FootworkJS (optionally on the specified node/element).
// start application (defaults to body element)
fw.start(); // document.body is the default element

NOTE: This is very similar to calling fw.applyBindings(). In fact they are identical except fw.start() is a simplified version which does not accept a viewModel / dataModel / router to bind, it simply causes FootworkJS to begin processing the DOM (and thus it will initialize and compose any <viewModel> / <dataModel> / <router> / <component> declarations it finds in the markup).

fw.applyBindings() will also cause Footwork to begin processing the DOM, but it also requires that you supply a viewModel / dataModel / router to bind with the root node.

fw.utils.guid()
Generate a random GUID (format: ########-####-####-####-############).
Returns: 'string' The name of the namespace channel 
// generate a random GUID
var guid = fw.utils.guid();
// guid === "629e4dfa-fd39-d756-b830-563af555a03f"
fw.sync( method, concern, options)

The function that Footwork calls every time it attempts to perform some sort of ajax request.

By default, it uses either jQuery.ajax or reqwest.ajax (ie: fw.ajax - using one of those depending on your particular environment) to make a RESTful JSON request and returns a Promise.

You can override it in order to use a different persistence strategy, such as WebSockets, XML transport, or Local Storage.

  • method - The CRUD method ('create', 'read', 'update', or 'delete').
  • concern - The dataModel or collection to sync.
  • options - All other jQuery-based request options.
Returns: Promise The request promise returned from fw.ajax 
var PeopleCollection = fw.collection.create({
  url: '/people'
});

fw.sync('read', people).done(function(response) {
  console.info('People returned', response);
});
fw.deferred()

Create a deferred object (can be used to create a promise).

Depending on the context of your usage, this will either be jQuery's Deferred() or D.js D() method.

Please refer to the relevant documentation for your Deferred() implementation for more information on how to use the returned deferred object.

Returns: Deferred Deferred object returned from the Deferred() method. 
var defer = fw.deferred();
defer.resolve({ success: true });

defer.then(function(resolutionValue) {
  console.info('Resolved with:', resolutionValue);
});
