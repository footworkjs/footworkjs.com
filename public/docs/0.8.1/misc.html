Utility

Utility functions / methods.

  • fw.start() - Start footwork / initialize application
  • fw.guid() - Generate a random GUID.

fw.start()

fw.start( targetElement )

Initialize / start footwork (optionally on the specified node/element):

// start application (defaults to body element)
fw.start();

// start application on #appContainer element
fw.start(document.getElementById('appContainer'));
NOTE: This is very similar to calling ko.applyBindings(). In fact they are identical except fw.start() is a simplified version which does not accept a viewModel to bind, it simply causes footwork to begin binding (and thus it will initialize any viewModel/router/component/etc definitions it finds in the markup). ko.applyBindings() does the same, but also requires that you supply a viewModel to bind.

fw.guid()

Generate a random GUID (format: ########-####-####-####-############):

// generate a random GUID
var guid = fw.guid();
